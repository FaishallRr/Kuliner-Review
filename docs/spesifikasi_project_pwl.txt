PROJECT 1 -- SISTEM PEMESANAN LAYANAN (SERVICE BOOKING)

A. Deskripsi Project
Sistem pemesanan layanan berbasis web yang memungkinkan pelanggan memesan jasa (bengkel, salon, klinik, laundry, dll) secara online. Dilengkapi dengan manajemen jadwal, konfirmasi booking, pembayaran via payment gateway (Midtrans Sandbox), dan notifikasi status.

B. Tech Stack & API Eksternal
· Framework: CodeIgniter 4 · Database: MySQL (via Migration & Seeder) · Payment Gateway: Midtrans Sandbox (free tier) -- https://sandbox.midtrans.com · Notifikasi: Email via PHPMailer + SMTP Gmail (free) · Opsional Webservice Client: RajaOngkir API (free tier) untuk estimasi ongkir jika ada pengiriman
produk, atau Google Calendar API untuk sinkronisasi jadwal

C. Fitur & Requirement Fungsional
Role: Admin · Kelola jenis layanan (CRUD: nama, deskripsi, harga, durasi, foto) · Kelola jadwal tersedia (tanggal, slot waktu, kapasitas) · Lihat semua booking (filter by status, tanggal, layanan) · Konfirmasi atau tolak booking · Kelola data teknisi/staff · Dashboard: statistik booking, pendapatan, layanan terpopuler · Manajemen user (aktif/nonaktif)
Role: Staff / Teknisi · Lihat jadwal tugas harian · Update status pengerjaan (proses, selesai) · Lihat riwayat pekerjaan pribadi
Role: Pelanggan · Register & login · Browse layanan yang tersedia · Pilih layanan, tanggal, dan slot waktu · Isi form booking (keterangan/catatan tambahan) · Pembayaran via Midtrans (transfer bank, QRIS, GoPay sandbox) · Lihat riwayat booking dan status pembayaran · Batalkan booking (jika belum dikonfirmasi) · Beri ulasan & rating setelah layanan selesai
Fitur Sistem · Filter proteksi route per role menggunakan CI4 Filter · Webhook Midtrans untuk update status pembayaran otomatis · Notifikasi email: konfirmasi booking, reminder H-1, update status · Upload foto untuk layanan dan bukti pengerjaan · API Endpoint: GET /api/services (daftar layanan publik), GET /api/booking-status/{id}

PROJECT 2 -- LOKASI KULINER & REVIEW JAJANAN

A. Deskripsi Project
Platform berbasis web untuk menemukan, menambahkan, dan mengulas tempat makan/jajanan di sekitar kampus atau area tertentu. Dilengkapi dengan geocoding otomatis, filter berdasarkan kategori & tag, dan sistem rating. Data lokasi terintegrasi dengan OpenStreetMap Nominatim API (gratis, unlimited).

B. Tech Stack & API Eksternal
· Framework: CodeIgniter 4 · Database: MySQL (via Migration & Seeder) · Geocoding API: OpenStreetMap Nominatim (free, unlimited) -- https://nominatim.openstreetmap.org · Map Display: Leaflet.js (open source, gratis) untuk tampilkan peta interaktif · Opsional: Foursquare Places API free tier atau Overpass API (OSM) untuk data POI tambahan · Upload: foto tempat makan dengan auto-resize menggunakan CI4 Image Library

C. Fitur & Requirement Fungsional
Role: Admin · Kelola kategori (warteg, kafe, street food, minuman, dll) · Kelola tag (halal, murah, AC, WiFi, parkir, dll) · Moderasi tempat kuliner (approve/reject submission dari user) · Moderasi review (hapus review yang tidak pantas) · Dashboard: statistik tempat, review, user aktif, rating tertinggi · Kelola data kuliner langsung (CRUD penuh)
Role: Kontributor (User Login) · Submit tempat makan baru (nama, alamat, deskripsi, kategori, tag, foto) · Geocoding otomatis: sistem auto-fetch koordinat dari Nominatim berdasarkan alamat · Tulis review dan beri rating (1-5 bintang) · Edit review sendiri (dalam 24 jam) · Upload hingga 3 foto per tempat · Simpan tempat favorit ke daftar pribadi · Tandai tempat sebagai "tutup permanen" (untuk divalidasi admin)
Role: Pengunjung (Tanpa Login) · Browse daftar kuliner dengan filter (kategori, tag, rating, jarak) · Lihat detail tempat, foto, peta lokasi (Leaflet) · Baca review dari kontributor · Cari berdasarkan nama atau alamat
Fitur Sistem · Integrasi Nominatim: otomatis convert alamat  lat/long saat submit tempat · Peta interaktif per tempat menggunakan Leaflet.js + tile OSM · Perhitungan rata-rata rating otomatis saat ada review baru · Webservice Server: GET /api/kuliner?lat=x&lng=y&radius=km (untuk poster/display kampus) · Filter route dengan CI4 Filter: submit dan review hanya untuk yang login

PROJECT 3 -- SISTEM INFORMASI MANAJEMEN PERPUSTAKAAN DIGITAL

A. Deskripsi Project
Sistem manajemen perpustakaan berbasis web yang mencakup pengelolaan koleksi buku, keanggotaan, peminjaman, pengembalian, dan denda. Dilengkapi notifikasi WhatsApp (WAHA API) dan email, integrasi Open Library API untuk auto-fill data buku, serta payment gateway (Midtrans) untuk pembayaran denda secara online.

B. Tech Stack & API Eksternal
· Framework: CodeIgniter 4 · Database: MySQL (via Migration & Seeder) · Book Metadata API: Open Library API (free, unlimited) --
https://openlibrary.org/api/books?bibkeys=ISBN:xxx&format=json
· Notifikasi WhatsApp: WAHA API (self-hosted atau cloud) -- kirim pesan WA ke anggota · Notifikasi Email: PHPMailer / CI4 Email Library + SMTP Gmail (free) · Payment Gateway: Midtrans Sandbox (free) -- pembayaran denda online

C. Fitur & Requirement Fungsional
Role: Admin / Pustakawan · Kelola koleksi buku (CRUD): input ISBN  auto-fetch judul, penulis, penerbit, cover dari Open Library
API
· Kelola eksemplar buku (kode eksemplar, kondisi, rak) · Kelola anggota (CRUD, cetak kartu anggota PDF) · Proses peminjaman dan pengembalian buku · Hitung denda otomatis berdasarkan hari keterlambatan · Tagih denda via Midtrans atau tandai lunas manual · Kirim notifikasi manual via WA atau email · Laporan: peminjaman aktif, buku terlambat, statistik bulanan · Dashboard: ringkasan stok, peminjaman aktif, denda belum dibayar
Role: Anggota (Member) · Register & login (dengan nomor telepon untuk WA notification) · Browse dan cari koleksi buku (judul, penulis, ISBN, kategori) · Lihat detail buku dan ketersediaan eksemplar · Ajukan peminjaman online (disetujui pustakawan) · Lihat riwayat peminjaman dan status pengembalian · Bayar denda online via Midtrans · Terima notifikasi: konfirmasi peminjaman, reminder H-3 & H-1 jatuh tempo, tagihan denda
Fitur Sistem · Auto-fill buku via Open Library API saat input ISBN · Kalkulasi denda otomatis: (tanggal kembali - tanggal jatuh tempo) × tarif/hari · Notifikasi WA otomatis: peminjaman disetujui, reminder jatuh tempo, denda · Notifikasi email: sama dengan WA sebagai backup · Payment Midtrans untuk denda, webhook update status otomatis
